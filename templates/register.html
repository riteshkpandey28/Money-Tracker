ABOUT

Money Tracker is Web based Application developed to help one track his/her income as well as expense. Money Tracker takes the data from the user and helps the user to efficiently manage his/her finance. This system also helps the user to visualize his/her financial status with the help of a graph.

Key Features:
  • User Friendly
  • Day wise tracking
  • Ample Visualization of financial status
  • Secure
